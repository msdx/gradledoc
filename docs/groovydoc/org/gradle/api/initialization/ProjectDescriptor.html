Package: org.gradle.api.initialization

[Java] Interface ProjectDescriptor

  • A ProjectDescriptor declares the configuration required to create and evaluate a Project.

    A ProjectDescriptor is created when you add a project to the build from the settings script, using Settings.include or Settings.includeFlat. You can access the descriptors using one of the lookup methods on the Settings object.
