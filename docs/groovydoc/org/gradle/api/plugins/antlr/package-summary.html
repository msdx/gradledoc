Package org.gradle.api.plugins.antlr Description

A Plugin for generating parsers from Antlr grammars.
  • Interface Summary 
    Interface Description
    AntlrSourceVirtualDirectory Contract for a Gradle "convention object" that acts as a handler for what I call a virtual directory mapping, injecting a virtual directory named 'antlr' into the project's various source.
  • Class Summary 
    Class Description
    AntlrPlugin A plugin for adding Antlr support to java projects.
    AntlrTask Generates parsers from Antlr grammars.
