            DeleteSpec

            public class Delete
extends org.gradle.api.internal.ConventionTask

            Deletes files or directories. Example:

             task makePretty(type: Delete) {
   delete 'uglyFolder', 'uglyFile'
   followSymlinks = true
 }
            Be default symlinks will not be followed when deleting files. To change this behavior call Delete.setFollowSymlinks with true. On systems that do not support symlinks, this will have no effect.
